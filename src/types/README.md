# types

Shared TypeScript type definitions.

## What it does

Central type definitions used across the entire codebase. Re-exports relevant types from `@polymarket/clob-client` and defines domain-specific types.

## Key types

- **Market / OrderBook / PriceLevel** — Market data structures
- **Position / OrderRequest / OrderResult** — Trading primitives
- **TradeSignal / StrategyMetrics / StrategyState** — Strategy I/O
- **BotConfig / RiskLimits** — Configuration
- **EventType / BotEvent** — Event bus typing
